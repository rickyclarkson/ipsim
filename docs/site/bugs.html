IPSim Bugs

Grave Bugs

FIXED: 2005-04-14: Some of the menu items don't generate log entries, which makes it difficult to mark the tests. These menu items are Add Default Route, Add Explicit Route, View/Edit Routing Table, View IP Addresses, Enable/Disable IP Forwarding, possibly among others.

FIXED: 2005-03-21: Rik Whittaker discovered that a network of three computers on one hub tends to show 'network unreachable'. The problem was that an incoming Ethernet packet's MAC address wasn't being examined, so all the computers were replying to one ping, causing strange race conditions.

FIXED: 2005-06-03: Adding more than one card to a computer seemed to add them all in the same place. A simple problem caused by List.indexOf not being generic.
