IPSim

IPSim is a network simulator, used in the University of Salford to help teach and assess Computer Science (and related courses) students on their static subnetting skills.

News

IPSim is due to be used for real assessments for the first time, and development continues to make it useful for related research topics.

Download

ipsim.zip - this is required if you are in the Newton laboratories. See the installation instructions.

ipsim.jar - this is required if you are on your own computer with a JRE installed.

To install the JRE, see the installation instructions.
